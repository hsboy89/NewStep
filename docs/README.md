# NewStep 문서

이 폴더에는 NewStep 프로젝트의 모든 문서가 포함되어 있습니다.

## 문서 목록

### 📋 [PRD.md](./PRD.md)
제품 요구사항 문서 (Product Requirements Document)
- 프로젝트 개요 및 목적
- 기능 요구사항 상세 설명
- 기술 스택
- 개발 로드맵

### 🏗️ [ARCHITECTURE.md](./ARCHITECTURE.md)
아키텍처 문서
- 프로젝트 구조
- 컴포넌트 계층
- 데이터 흐름
- 상태 관리 구조
- 주요 유틸리티 함수 설명

### 🔌 [API.md](./API.md)
API 문서
- 외부 API 사용법 (rss2json, Dictionary API, Web Speech API)
- 내부 유틸리티 함수 API
- 요청/응답 형식
- 에러 핸들링

### 👨‍💻 [DEVELOPMENT.md](./DEVELOPMENT.md)
개발 가이드
- 개발 환경 설정
- 코드 컨벤션
- 새로운 기능 추가 방법
- 디버깅 가이드
- 빌드 및 배포

## 빠른 시작

새로운 개발자를 위한 빠른 가이드:

1. **프로젝트 이해**: [PRD.md](./PRD.md) 읽기
2. **구조 파악**: [ARCHITECTURE.md](./ARCHITECTURE.md) 읽기
3. **개발 시작**: [DEVELOPMENT.md](./DEVELOPMENT.md) 따라하기
4. **API 참조**: 필요시 [API.md](./API.md) 참고

## 문서 업데이트

프로젝트 변경사항이 있을 때 관련 문서를 함께 업데이트해주세요:

- **기능 추가/변경**: PRD.md, ARCHITECTURE.md 업데이트
- **API 변경**: API.md 업데이트
- **개발 프로세스 변경**: DEVELOPMENT.md 업데이트

---

**참고**: 프로젝트 루트의 [README.md](../README.md)는 사용자를 위한 간단한 사용 가이드입니다. 개발자용 상세 문서는 이 폴더를 참고하세요.
